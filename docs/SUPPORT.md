# Support - Guess It All

## Questions fréquentes (FAQ)

### Comment jouer ?

1. **Configurez la partie** : choisissez le nombre d'équipes et de joueurs
2. **Formez les équipes** : attribuez les joueurs manuellement ou aléatoirement
3. **Ajoutez les mots** : saisissez vos propres mots ou utilisez les catégories prédéfinies
4. **Jouez les 3 manches** :
   - **Manche 1** : Décrivez le mot librement (sans le prononcer)
   - **Manche 2** : Un seul mot pour faire deviner
   - **Manche 3** : Mimez sans parler

### L'application ne se lance pas

- Vérifiez que votre appareil est compatible (Android 5.0 minimum)
- Essayez de redémarrer l'application
- En dernier recours, désinstallez et réinstallez l'application

### Mes données ont disparu

Les données de jeu (paramètres, préférences) sont stockées localement sur votre appareil. Elles peuvent être perdues si :
- Vous désinstallez l'application
- Vous effacez les données de l'application dans les paramètres Android

### Puis-je jouer hors-ligne ?

Oui ! Guess It All fonctionne entièrement hors-ligne. Aucune connexion internet n'est nécessaire.

### Comment ajouter mes propres mots ?

Lors de la configuration de la partie, choisissez l'option "Mots personnalisés". Chaque joueur pourra alors saisir ses propres mots à faire deviner.

### L'application contient-elle des publicités ?

Non, Guess It All est 100% sans publicité.
