# theme/

Theme visuel de l'application.

## Fichier unique : `app_theme.dart`

### `AppColors`
- **Background** : noir `#0A0A0A`
- **Primary** : rose `#FF3B7F`
- **Secondary** : cyan `#00E5FF`
- **Etats** : success (vert), error (rouge), warning (orange)
- **Gris** : 5 niveaux
- **Equipes** : 4 couleurs (rose, cyan, violet, orange)

### `AppTheme.darkTheme`
- Theme sombre Material avec polices Bangers (titres) et Poppins (corps)

### `AppTextStyles`
- `title` / `subtitle` : Bangers avec ombres
- `body` / `label` : Poppins
